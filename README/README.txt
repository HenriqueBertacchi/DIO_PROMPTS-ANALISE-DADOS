
# Análise de Relatórios de Vendas com IA

## Sobre o Projeto  
Este projeto explora o uso de prompts com ferramentas de inteligência artificial para analisar relatórios de vendas, identificar padrões e gerar insights valiosos para apoiar decisões estratégicas no negócio.

## Objetivos  
- Aplicar conceitos de IA para análise de dados comerciais  
- Extrair insights relevantes a partir dos dados de vendas  
- Documentar de forma clara as análises e resultados  
- Utilizar o GitHub para compartilhar o projeto  

## Análises Realizadas  

### 1. Faturamento por Produto  
O produto **NEW MEGANIUM RG 40XXV** é o mais vendido e principal responsável pelo faturamento da linha.

| NEW MEGANIUM RG 40XXV  | 41                 | 4100              |
| NEW MEGANIUM RG35XX    | 36                 | 3240              |
| MEGANIUM RG353M        | 29                 | 3190              |

## Considerações Finais  
- RG 40XXV é o carro-chefe da linha de produtos.  
- O mercado canadense deve ser prioridade para estratégias comerciais.  
- Descontos estão sendo usados como ferramenta de vendas para alguns produtos.  
- Campanhas no mês de agosto são eficazes e devem ser consideradas para futuras ações.
